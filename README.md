📊 Customer Behavior Analysis
📌 Project Overview

This project focuses on analyzing customer behavior to extract meaningful insights from raw data. The analysis was performed using Python, followed by database querying and dashboard visualization.

🎯 Objectives
Understand customer purchasing patterns
Identify trends in customer behavior
Extract actionable insights using data analysis

🛠️ Tools & Technologies
Python (Pandas)
PostgreSQL
Power BI
Jupyter Notebook

🔄 Project Workflow

1️⃣ Data Cleaning & Preprocessing
Removed missing values
Eliminated duplicate records
Standardized data formats using Pandas


2️⃣ Exploratory Data Analysis (EDA)
Analyzed customer purchase behavior
Identified patterns and trends
Performed data exploration using Pandas operations


3️⃣ Database Integration
Imported cleaned dataset into PostgreSQL
Created structured tables for analysis


4️⃣ SQL Analysis
Queried data to extract insights such as:
Customer segmentation
Purchase frequency
Product trends


5️⃣ Dashboard Creation
Developed an interactive dashboard using Power BI
Visualized key insights and metrics


📈 Key Insights
Identified high-value customers
Found frequently purchased products
Observed patterns in customer purchases
